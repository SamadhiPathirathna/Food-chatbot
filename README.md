# 🍽️ Food Suggestion Chatbot

A smart and friendly food recommendation chatbot that helps users discover meal suggestions for breakfast, lunch, dinner, snacks, or a full-day meal plan — with nutrition info, voice input, favorites, and light/dark mode. Built with HTML, CSS, JavaScript (frontend) and Flask (backend), powered by the Spoonacular API.

---

## ✨ Features

- 🍳 Get meal suggestions based on your selected meal type (Breakfast, Lunch, Dinner, Snack, or Daily Plan)
- 🥗 Choose your preference: Vegetarian or Non-Vegetarian
- 🍱 Select a food category: Healthy or Comfort
- 📊 View nutritional info (calories, protein, carbs, fat)
- ⭐ Save favorites and view them anytime
- 🔁 Get another suggestion with one click
- 🌓 Toggle between Light & Dark mode
- 🎙️ Voice command support (if supported by your browser)

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Python (Flask)
- **API**: [Spoonacular Food API](https://spoonacular.com/food-api)
- **Browser API**: Web Speech API (for voice input)
- **Storage**: LocalStorage for saving favorites and theme preference
